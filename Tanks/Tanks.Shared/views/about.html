About

Wild Wild Tanks is a demo project, showing how you can use JS itself and popular JS libraries for building Windows Store Apps. It uses AngularJS in conjunction with WinJS to build UI layer of the game.

On the other side, heavy things you can delegate to compiled code, C# or C++. For instance, core of the Wild Wild Tanks was programmed in C#, through portable library, and then referenced by Windows and Windows.Phone presentation layer projects.

Full source code of this project you can find on GitHub.
